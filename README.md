# Mario.JACK

Descripción
---------

`Mario.JACK` es una versión educativa de un juego tipo plataforma/Tetris híbrido llamada "TetriPath" escrita en Jack para usar con el IDE de nand2tetris. El proyecto combina una fase de construcción (colocación de piezas tipo Tetris) con una fase de carrera donde el jugador controla al personaje para llegar a la meta.

¿Cómo usarlo?
----------------

1. Clonar el proyecto

	Ejecuta en una terminal:

	```bash
	git clone https://github.com/Peejhay55/Mario.JACK.git
	```

2. Acceder al IDE de nand2tetris (sección Virtual Machine - VM)

	- URL: https://nand2tetris.github.io/web-ide/vm

3. Cargar el juego

	- En el lateral izquierdo del IDE, haz clic en el icono de carpeta.
	- Se abrirá el navegador de archivos; navega hasta la carpeta donde clonaste el proyecto y selecciónala para cargar los archivos en el IDE.

4. Ajustes recomendados para mejor experiencia (opcionales)

	a) Sobre el recuadro blanco, a la derecha de "Screen", selecciona la opción `x2` para escalar la pantalla.


	b) En el panel izquierdo de la VM, arrastra la barra de velocidad hasta `fast`.

5. Correr el juego

	- En la sección lateral donde aparece "VM Code", pulsa el botón `run` o el botón azul central para ejecutar la VM y jugar.

Disfruta y avísanos si quieres que añada instrucciones para ejecutar pruebas o compilar desde línea de comandos.
